HTML-Tag

HTML-Tag Name Description
<html> HTML The root of the document
<a> Anchor Tag Creates a hyperlink. Links to a external URL
<body> Body The document's body is required
<br> Break Single line text break
<div> Division A block in a document used mainly to layout sections in the design of the page
<form> Form Form for user input which wraps the input fields
<h1-4> Headings HTML headings. H1 is the main heading of the page
<head> Head Information about the document. It contains meta tags, and other decalarations
<img> Image Image on the page
<input> Input Field Create a type of input field for a User to enter data
<link> Link Links to an external file
<p> Paragraph Paragraph for entering text
<script> Script Used for Client-sided scriping
